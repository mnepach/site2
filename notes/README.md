## Описание проекта
Мобильный сайт для продажи товаров для активного отдыха в Беларуси, включая снаряжение для рыбалки, походов, ножи и одежду

## Технические характеристики
**Фреймворк**: React 18 с Babel для трансформации JSX
**Стили**: Tailwind CSS с кастомными CSS-переменными для цветовой схемы
**Шрифт**: Inter (400, 500, 600, 700)
**Иконки**: Lucide Static для минималистичных иконок
**Адаптивность**: Mobile-first подход, оптимизирован для экранов от 320px
**Обработка ошибок**: Реализована через `ErrorBoundary` и try-catch в каждом компоненте
**Изображения**: Оптимизированы через Unsplash с параметрами `auto=format&fit=crop&w=400&q=80`

## Структура сайта
1. **Header**: Логотип "Лесной Край" с иконкой сосны и кнопки корзины/профиля.
2. **Hero**: Главное предложение с фоновым изображением леса, призывом к действию и кнопкой перехода к товарам.
3. **ProductGrid**: Категоризированный список товаров с фильтрацией по категориям.
4. **ProductCard**: Карточка товара с изображением, названием, описанием, ценой и метками "Новинка" или скидки.
5. **Footer**: Контактная информация, ссылки на доставку, возврат, гарантию и информацию о компании.
6. **products**: Описание самих товаров

## Цветовая схема
- **Основной**: #2d4a3e (тёмно-зелёный) — для фона хедера, кнопок и логотипа
- **Вторичный**: #8b7355 (коричнево-бежевый) — для ховера кнопок
- **Акцент**: #d4af37 (золотистый) — для цен и метки "Новинка"
- **Акционная цена**: #ef4444 (красный, `bg-red-500`, `text-red-500`) — для меток скидок
- **Фон**: #f8f6f0 (кремовый, `--bg-cream`) — для основного фона
- **Текст**: #1a1a1a (тёмно-серый, `--text-dark`) — основной текст, #666666 (серый, `--text-light`) — второстепенный текст
- **Границы**: #e0ddd4 (светло-бежевый, `--border-color`) — для границ карточек и хедера
